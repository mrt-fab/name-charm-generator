# つながるネームチャーム

テキストを入力すると、**文字同士が可動ジョイントでつながった3Dプリント用ネームキーチャーム**のSTLデータを生成するWebアプリです。ブラウザだけで完結し、サーバーへのデータ送信は一切ありません。

Bambu Lab P2S（0.4mmノズル・PLA・レイヤー0.2mm）を想定していますが、一般的なFDMプリンターで印刷できます。

## 使い方

1. 名前を入力（ひらがな・カタカナ・英数字）
2. フォント・サイズ・色数・金具を選ぶ
3. **生成する** を押して3Dプレビューを確認
4. STL（複数色はZIP）をダウンロードしてスライサーへ

### 複数色STLのBambu Studioでの読み込み

1. ZIPを展開し、すべてのSTLを同時にドラッグ&ドロップ
2. 「1つのオブジェクトの複数パーツとして読み込みますか？」→ **はい**
3. パーツごとにフィラメント（色）を割り当て（color1が最下層＝ビルドプレート側）

色の境界は印刷レイヤーに平行な水平面なので、AMSがなくても「指定レイヤーでフィラメント交換」で印刷できます。

## 印刷推奨設定

- レイヤー 0.2mm ／ サポートなし ／ PLA
- ジョイントが固い場合はクリアランス「ゆるめ」で再出力
- 初回はビルドプレートの1層目が潰れすぎないよう調整してください（関節固着の主原因）

## フォントについて

同梱フォントはすべて SIL Open Font License 1.1（商用利用可）です。ライセンス文は `fonts/` を参照してください。

**手元のフォントファイル（.ttf / .otf）を読み込む機能**もあります。読み込んだフォントはブラウザ内（IndexedDB）にのみ保存され、外部送信・再配布はされません。フォント自体のライセンス（Web利用・立体物への利用可否）は各自でご確認ください。

## 開発

```bash
python3 dev_server.py 8793
```

で `http://localhost:8793` から開発版を起動できます（キャッシュ無効化付き静的サーバー）。技術的な設計メモはリポジトリ外の運用ドキュメントにあります。
