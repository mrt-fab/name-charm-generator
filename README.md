# つながるネームチャーム

テキストを入力すると、**文字同士が可動ジョイントでつながった3Dプリント用ネームキーチャーム**のSTLデータを生成するWebアプリです。ブラウザだけで完結し、サーバーへのデータ送信は一切ありません。

Bambu Lab P2S（0.4mmノズル・PLA・レイヤー0.2mm）を想定していますが、一般的なFDMプリンターで印刷できます。

**▶ 公開URL: https://mrt-fab.github.io/name-charm-generator/**（社内共有用）

## 使い方

1. 名前を入力して **このテキストで生成** を押す（名前以外の設定は動かした瞬間にリアルタイム反映）
2. プリセットチップやランダムから始めて、フォント・サイズ・色・金具を調整
3. STL（複数色はZIP）をダウンロードしてスライサーへ
4. **この設定のリンクをコピー** で、同じ構成を誰にでもURLで共有できます

### 金具・チェーン

- **ループ穴**: 市販の金具・ストラップを通す穴。金具「なし」でもこれが付きます
- **チェーン**: 楕円リンクを交互に傾けて絡ませた本物のチェーン構造（サポートレスで一体印刷）
- **スナップリング**: フック＆アイ式のクラスプ。リングを軽くひねると開閉できます（一体印刷）
- **カラビナ**: ピボットゲート式。ねじロック用のローレットノブは**別パーツ**として同じSTLに並んで出力されるので、印刷後にポストへねじ込んでください

### 複数色STLのBambu Studioでの読み込み

1. ZIPを展開し、すべてのSTLを同時にドラッグ&ドロップ
2. 「1つのオブジェクトの複数パーツとして読み込みますか？」→ **はい**
3. パーツごとにフィラメント（色）を割り当て（color1が最下層＝ビルドプレート側）

色の境界は印刷レイヤーに平行な水平面なので、AMSがなくても「指定レイヤーでフィラメント交換」で印刷できます。

## 印刷推奨設定

- レイヤー 0.2mm ／ サポートなし ／ PLA
- ジョイントやカラビナのねじが固い場合はクリアランス「ゆるめ」で再出力
- 1層目が潰れすぎると関節が固着します（エレファントフット補正 約0.15mmを推奨）

## フォントについて

同梱フォント10種はすべて SIL Open Font License 1.1（商用利用可）です。配信用にサブセット化しており、原本は `fonts/src/`、ライセンス文は `fonts/licenses/` にあります。

**手元のフォントファイル（.ttf / .otf）を読み込む機能**もあります。読み込んだフォントはブラウザ内（IndexedDB）にのみ保存され、外部送信・再配布はされません。フォント自体のライセンス（Web利用・立体物への利用可否）は各自でご確認ください。

## 開発

```bash
python3 dev_server.py 8793
```

で `http://localhost:8793` から開発版を起動できます（ESモジュールがChromeにキャッシュされないようno-storeヘッダー付きで配信する静的サーバー）。

フォントサブセットの再生成:

```bash
python3 -m venv .venv && .venv/bin/pip install fonttools
.venv/bin/python scripts/subset_fonts.py
```

開発フローはGitHub Flow（featureブランチ→PR→main）。mainがそのままGitHub Pagesで配信されます。
